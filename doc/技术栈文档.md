# 在线问卷调查与数据分析系统 - 技术栈文档

## 一、后端技术栈

### 核心框架
- **Java 1.8** - 开发语言
- **Spring Boot 2.6.13** - 核心框架
- **Spring MVC** - Web层框架
- **MyBatis Plus 3.5.x** - ORM框架
- **Spring Security** - 安全框架
- **Spring AOP** - 面向切面编程

### 数据存储
- **MySQL 8.0+** - 关系型数据库
- **Redis 6.0+** - 缓存数据库（会话存储、数据缓存、限流控制）

### 认证与授权
- **JWT** - 无状态身份认证
- **Spring Security JWT** - JWT集成
- **BCrypt** - 密码加密

### 数据处理
- **Jackson** - JSON序列化/反序列化
- **Apache POI** - Excel文件处理
- **EasyExcel** - 高性能Excel处理
- **iText / Flying Saucer** - PDF文件生成（数据报告、分析报告导出）

### 工具类库
- **Lombok** - 简化Java代码
- **Hutool** - Java工具类库
- **Apache Commons** - 通用工具类

### 接口文档生成
- **Swagger / Knife4j** - API接口文档自动生成工具
  - 自动生成RESTful API文档
  - 支持在线测试接口
  - 提供交互式API文档界面

### 日志
- **Logback** - 日志框架
- **SLF4J** - 日志门面

### 构建工具
- **Maven 3.6+** - 项目构建与依赖管理

### 文件上传处理
- **Spring Boot Multipart** - 文件上传支持（内置）
- **本地文件系统** - 文件存储在后端项目根目录下的upload文件夹

### 其他
- **Validation API** - 数据校验
- **Druid** - 数据库连接池

---

## 二、前端技术栈

### 核心框架
- **JavaScript** - 脚本语言
- **Vue 3.x** - 渐进式JavaScript框架
- **Vite** - 前端构建工具

### UI组件库
- **Element Plus** - Vue 3 UI组件库
- **@element-plus/icons-vue** - Element Plus图标库

### 状态管理
- **Pinia** - Vue 3状态管理库

### 路由
- **Vue Router 4.x** - 前端路由管理

### HTTP请求
- **Axios** - HTTP客户端
- **axios拦截器** - 请求/响应拦截

### 数据可视化
- **ECharts 5.x** - 数据可视化图表库
- **vue-echarts** - ECharts的Vue组件封装

### 表单处理
- **@vueuse/core** - Vue组合式API工具集

### 富文本编辑
- **Quill** - 富文本编辑器

### 拖拽功能
- **Vue Draggable / SortableJS** - 拖拽排序

### 工具库
- **Lodash-es** - JavaScript工具库
- **Day.js** - 日期处理库
- **qrcode.js** - 二维码生成
- **html2canvas** - HTML转图片
- **jsPDF** - PDF文件生成（前端PDF导出，可选）
- **js-file-download** - 文件下载

### 样式处理
- **Sass / Less** - CSS预处理器
- **PostCSS** - CSS后处理器
- **Autoprefixer** - CSS自动添加浏览器前缀

### 代码规范
- **ESLint** - JavaScript代码检查
- **Prettier** - 代码格式化
