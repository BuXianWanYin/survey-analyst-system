# 在线问卷调查与数据分析系统前端

## 项目简介

在线问卷调查与数据分析系统前端，基于Vue 3 + Vite + Element Plus开发。

## 技术栈

- Vue 3.x
- Vite 5.x
- Element Plus
- Pinia
- Vue Router 4.x
- Axios
- ECharts 5.x

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```
